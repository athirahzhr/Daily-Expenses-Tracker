# 💰 Daily Expenses Tracker

A web-based expense management system built using Google Apps Script and Google Sheets.

## Features

- Add Daily Expenses
- Weekly Spending Limit
- Expense History
- Daily, Weekly & Monthly Reports
- Interactive Bar Chart
- Pie Chart by Category
- Weekly Expense Comparison
- Print Report
- Responsive Interface

## Technologies

- Google Apps Script
- HTML5
- CSS3
- JavaScript
- Google Sheets
- Chart.js
